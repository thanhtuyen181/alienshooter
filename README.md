# Alienshooter

**Introduction to Embedded Systems Course Project**

## Overview

**Alienshooter** is a project developed for a **Introduction to Embedded Sytems** course at Tampere University.  
Thoughout this project, I work with my classmate Zannatul Ferdous to create a 2D game using C and ARM assembly where players control a ship to shoot aliens on an embedded system.
